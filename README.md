# 📌 Project Overview

This project is an interactive Tableau dashboard designed to analyze hotel performance using booking, revenue, customer, and satisfaction data.

The dashboard enables hotel managers to monitor KPIs, identify revenue opportunities, understand customer behavior, and make data-driven business decisions through a fully interactive reporting solution.

---

# 🎯 Business Problem

Hotels generate large volumes of booking and guest data every day. However, without an effective analytics solution it becomes difficult to:

- Track revenue performance
- Compare booking channels
- Identify profitable customer segments
- Monitor guest satisfaction
- Improve occupancy and revenue
- Support strategic decision-making

This dashboard addresses these challenges by providing an executive-level analytical solution.

# 📊 Dashboard 1 – Executive Performance Dashboard

![Executive Dashboard](assets/Executive.png)

### Key Features

- Revenue KPIs
- Customer Satisfaction Score
- Guest Count
- Revenue by Booking Channel
- Stay Duration Distribution
- Revenue Composition Analysis

### Key Insights

- Online Travel Agencies generate the highest revenue.
- Corporate bookings contribute consistent revenue.
- Customer satisfaction averages **3.0 / 4**.
- Revenue is primarily driven by room bookings.

---

# 📊 Dashboard 2 – Guest Insights & Segmentation

![Guest Insights](assets/Insights.png)

### Key Features

- Customer Segment Analysis
- Revenue Driver Matrix
- Guest Age Distribution
- Revenue by Country
- Average Stay Analysis
- Guest Satisfaction by Age Group

### Business Insights

- Budget Guests represent the largest customer segment.
- Additional spending contributes approximately **6.4%** of total revenue.
- UK generates the highest hotel revenue.
- Online Travel Agency is the highest-performing booking source.
- Single Rooms record the longest average stay.

# 📊 Dashboard 3 – Executive Recommendations

![Recommendations](assets/Recommendation.png)

The dashboard concludes with business recommendations derived from analytical findings.

### Recommendations

- Expand marketing investment in high-revenue countries.
- Promote premium room offerings through targeted campaigns.
- Increase focus on Online Travel Agencies.
- Encourage additional guest spending through bundled services.
- Develop loyalty programs for high-value guests.
